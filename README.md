# Tazz Video Downloader

Aplicativo em batch para Windows que baixa vídeos ou áudios do YouTube usando yt-dlp. Permite escolher formato (mp4/mp3), definir pasta de download, registrar histórico (com opções para reexecutar, abrir, copiar link, apagar registro e deletar arquivo) e alternar exibição dos últimos 10 ou todos os downloads.
